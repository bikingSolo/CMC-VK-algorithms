# Домашнее задание № 1

Домашнее задание направлено на закрепление навыков работы с массивами и построения алгоритмов линейной сложности.
В рамках выполнения необходимо реализовать решения с использованием техник in-place модификации, слияния и сортировки без дополнительных аллокаций, а также провести анализ временной и пространственной сложности.
Особое внимание уделяется выбору оптимального алгоритмического подхода и обоснованию эффективности решения.

**Цель домашнего задания**\
Закрепить навыки работы с массивами и алгоритмами линейной сложности.
Отработать техники in-place модификации данных, слияния и сортировки без дополнительных аллокаций.
Научиться анализировать временную и пространственную сложность решений и выбирать оптимальный алгоритмический подход.

**Оценивание**\
За каждую решенную задачу начисляется 2 балла, итого максимально можно получить 14 баллов.
 
## 1. Развернуть часть массива

Дан массив целых чисел.\
Необходимо повернуть (сдвинуть) справа налево часть массива, которая указана параметром k.\
Решение должно работать за линейное время без дополнительных аллокаций.\
**Пример:**\
Исходный массив: 1, 2, 3, 4, 5, 6, 7\
k = 3\
Результат: 5, 6, 7, 1, 2, 3, 4\
**Что нужно сделать:**\
Реализуйте функцию, которая выполняет циклический сдвиг части массива in-place. Объясните, почему ваше решение работает за O(n) и не использует дополнительную память.
 
## 2. Слияние двух отсортированных массивов
Дано два отсортированных массива.\
Необходимо написать функцию, которая объединит эти два массива в один отсортированный.\
**Что нужно сделать:**\
Реализуйте алгоритм слияния, используя подход двух указателей. Убедитесь, что итоговый массив сохраняет корректный порядок элементов.
 
## 3. Слияние двух отсортированных массивов (без дополнительных аллокаций)

Дано два отсортированных массива.\
Первый массив имеет размер результирующего массива, а его конец заполнен нулями.\
Объединить массивы необходимо без создания третьего массива.\
**Аналог задачи:** Merge Sorted Array (LeetCode)\
<https://leetcode.com/problems/merge-sorted-array/description/>\
**Что нужно сделать:**\
Напишите функцию, которая объединяет массивы, заполняя первый массив с конца. Поясните, почему заполнение с конца позволяет избежать перезаписи данных.
 
## 4. Сортировка массива из 0 и 1

Дан массив, содержащий только 0 и 1.\
Напишите функцию, которая сортирует его так, чтобы все нули оказались в начале, а единицы — в конце.
Решение должно быть in-place.\
**Что нужно сделать:**\
Реализуйте алгоритм, который проходит массив один раз и переставляет элементы без создания нового массива. Объясните, почему ваше решение имеет линейную сложность.
 
## 5. Задача флага Нидерландов

Дан массив, состоящий из 0, 1 и 2.\
Необходимо отсортировать его за линейное время.\
**Аналог задачи:** Sort Colors (LeetCode)
<https://leetcode.com/problems/sort-colors/description/>
Реализуйте алгоритм трёх указателей (Dutch National Flag). Кратко опишите, как поддерживаются границы для 0, 1 и 2 во время прохода по массиву.
 
## 6. Передвинуть чётные числа вперёд

Дан неотсортированный массив целых чисел.\
Необходимо перенести в начало массива все чётные числа, сохраняя их относительный порядок.\
Пример:\
Ввод: [3, 2, 4, 1, 11, 8, 9]\
Вывод: 2, 4, 8, 1, 11, 3, 9\
**Что нужно сделать:**\
Реализуйте стабильное перераспределение элементов, сохраняя порядок чётных чисел. Объясните, за счёт чего сохраняется относительный порядок элементов.
 
## 7. Нули в конец

В школе прошёл экзамен по математике. Несколько человек списали решения и получили 0 баллов.\
Дан массив оценок, среди которых есть 0.\
Необходимо перенести все нули в конец массива, чтобы такие ученики оказались в конце списка.\
Порядок остальных элементов должен сохраняться.\
**Что нужно сделать:**\
Реализуйте алгоритм, который за один проход переносит все ненулевые элементы вперёд. Поясните, почему ваше решение является стабильным и работает за O(n).
