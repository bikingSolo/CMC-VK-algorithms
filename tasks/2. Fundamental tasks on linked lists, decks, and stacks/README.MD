# Домашнее задание № 2

Домашнее задание направлено на **закрепление навыков работы с односвязными списками, строками и использованием техники двух указателей для решения задач за линейное время**. Вам предстоит реализовать разворот, слияние и удаление элементов в списках, а также задачи на проверку палиндромов, исходной строки и поиск пар в массиве — все in-place с анализом сложности алгоритмов.

**Цель домашнего задания**

* Закрепить навыки работы с односвязными списками и строками, а также освоить технику двух указателей для решения задач за линейное время.
* Научиться изменять структуры данных in-place, анализировать сложность алгоритмов и выбирать оптимальную стратегию обхода.

**Оценивание**\
За каждую решенную задачу начисляется 2 балла, итого максимально можно получить 16 баллов.
 

## 1. Развернуть односвязный список

Необходимо написать функцию, которая принимает на вход односвязный список и разворачивает его.\
**Что нужно сделать:**\
Реализуйте разворот списка in-place, изменяя ссылки между узлами. Объясните, как меняются указатели на каждом шаге и почему алгоритм работает за O(n).
 
## 2. Найти середину списка

Дан односвязный список.\
Необходимо найти его середину за O(n) без дополнительных аллокаций\
**Что нужно сделать:**\
Используйте метод двух указателей (медленный и быстрый). Поясните, почему быстрый указатель позволяет определить середину за один проход.
 
## 3. Удалить элемент из односвязного списка

Необходимо написать функцию, которая принимает односвязный список и целое число val. Нужно удалить узел, значение которого равно val.\
**Что нужно сделать:**\
Реализуйте удаление узла без создания нового списка. Объясните, как обрабатываются случаи удаления головы списка и внутренних элементов.
 
## 4. Является ли одна строка исходной для другой

В исходную строку добавили некоторое количество символов.\
Необходимо определить, была ли строка a исходной для строки b.\
**Что нужно сделать:**\
Реализуйте алгоритм сравнения строк с использованием указателей. Объясните, как вы определяете, что вторая строка получена добавлением символов к первой.
 
## 5. Метод двух указателей (поиск пары)

Дан отсортированный массив чисел и целевое число target.\
Необходимо определить, существуют ли два элемента, сумма которых равна target.\
**Что нужно сделать:**\
Реализуйте решение с использованием двух указателей (один в начале, другой в конце массива). Поясните, почему такой подход работает за O(n).
 
## 6. Является ли слово палиндромом?

Напишите функцию, которая принимает строку и возвращает true, если она является палиндромом.\
В противном случае вернуть false.\
Палиндром — слово или текст, одинаково читающиеся в обоих направлениях.\
**Что нужно сделать:**\
Реализуйте проверку с помощью двух указателей. Объясните, как сравниваются символы и почему алгоритм работает за линейное время.
 
## 7. Метод двух указателей (удаление дубликатов)

Дан отсортированный массив.\
Необходимо удалить дубликаты in-place так, чтобы каждый элемент встречался только один раз.\
**Что нужно сделать:**\
Реализуйте алгоритм с двумя указателями (медленный и быстрый). Поясните, как обеспечивается перезапись дубликатов без использования дополнительной памяти.
 
## 8. Слияние двух отсортированных списков

Написать функцию, которая принимает два отсортированных односвязных списка и объединяет их в один отсортированный список.\
Затраты по памяти должны быть O(1).\
Входные данные:\
list1 = [3, 6, 8]\
list2 = [4, 7, 9, 11]\
Выходные данные:\
[3, 4, 6, 7, 8, 9, 11]\
**Что нужно сделать:**\
Реализуйте слияние, изменяя только ссылки между существующими узлами. Объясните, почему алгоритм работает за O(n + m) и не требует дополнительной памяти.
